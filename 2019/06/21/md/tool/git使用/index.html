发表于 2019-06-21 更新于 2019-08-20

解决git pull 和 git push 每次都需要输入密码的问题

git config --global credential.helper store
之后再次执行git push 或者git pull这时候还需要输入用户名和密码,下次就不需要了

回滚远程提交

git reset –hard HEAD^
git push –force
